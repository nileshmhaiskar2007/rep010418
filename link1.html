Python Shallow copy

				Shallow Copy
				When we create an object in Python, then assign that object to another variable, Python doesn't copy the object. Instead, it copies only a reference to the object.

Python copy an object

				How to copy an object
				>>>customer = copy.copy(person)
				>>> husband = person[:] 
				>>> husband 
				['name', ['savings', 100.0]] 
				>>> wifey = list(person) 
				>>> wifey 
				['name', ['savings', 100.0]] 
				>>> [id(x) for x in person,husband,wifey] 
				[140502587231840, 140502587309520, 140502587309952] 

				>>>wifey[1][1]=50.0
				>>>wifey
				['name', ['savings', 50.0]]
				>>>husband
				['name', ['savings', 50.0]]
				>>>person
				['name', ['savings', 50.0]]
